Quickstart

Get started in under 60 seconds. No apps to install, no account needed.

01
Save the OURIN Number
Add +91 6000 570 671 to your contacts or tap the WhatsApp link to open a chat directly.
02
Send Any Message
Open the chat and send any message. OURIN responds instantly to confirm it's active.
03
Type .menu
Shows all 35 categories. Use .menucat <category> — e.g. .menucat ai or .menucat rpg.
04
Add to a Group (Optional)
Add OURIN to any group to unlock 109 group management commands including anti-spam, welcome, polls and admin tools.
💬
Default prefix is . — Premium users can set a custom prefix with .setprefix. Commands work in both personal and group chats.

Self-Hosting

Lifetime plan subscribers get full source code and can self-host on any Node.js server. The team provides 1-on-1 VPS deployment help.

Requirements

system
# Node.js 18+
node --version  # >= 18.0.0
npm  --version  # >= 9.0.0
# Ubuntu 20.04+ or Debian 11+ VPS recommended

Setup

bash
# Clone repo (Lifetime plan only)
$ git clone https://github.com/ashishmandal029/ourin-bot
$ cd ourin-bot && npm install
# Scan QR code with WhatsApp to link your number
$ npm start
✅ OURIN Bot v2.5.0 — Connected!
⚠️
Source code access is for Lifetime plan subscribers only. Contact the team on WhatsApp or Telegram.
