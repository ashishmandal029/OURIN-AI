Introduction

OURIN is a next-generation WhatsApp automation bot built on the Baileys framework. It was created in 2023 and has evolved into a comprehensive platform offering over 500 commands spanning AI, media tools, group management, interactive message formats, games, and more.

Unlike basic bots, OURIN treats WhatsApp as a full application platform — enabling formatted table messages, Google Sheets integration, multi-card carousels with interactive buttons, and real-time AI responses, all natively inside any WhatsApp conversation.

Self-Hosting

Lifetime plan subscribers get full source code access and can self-host OURIN on any Node.js server, including a VPS. The team provides 1-on-1 deployment assistance.

Requirements

system
# Node.js 18+

bash

$ cd ourin-bot
$ npm install

Smart Tables

OURIN's table feature renders beautiful Unicode table messages in WhatsApp — perfect for sharing student data, product lists, schedules, reports, or any structured information.

Manual Table — .sendtable

Format: pipe-separated rows. First 3 lines are header (title, subtitle, footer). Fourth line is column names. Subsequent lines are data rows.

whatsapp
❯ .sendtable @me
|📋 Student Report
|Class 12 Results 2026
|Official Data
|Name,Subject,Marks,Grade
|Ashish,Math,95,A+
|Riya,Science,88,A
|Arjun,English,76,B+

✅ Table sent to your chat!

Google Sheets Table — .sendgtable

Pull data live from any public Google Sheet and send it as a formatted table. Great for teachers, team leads, and anyone managing data collaboratively.

whatsapp
❯ .sendgtable @me
|https://docs.google.com/spreadsheets/d/18H3OWu.../edit
|📊 Student List
|Powered by OURIN

⏳ Fetching from Google Sheet...
📊 Found 7 rows, 10 columns
✅ Table sent! 7 rows displayed
📄
Your Google Sheet must be set to Anyone with the link can view. The bot fetches the first sheet by default.
CommandDescriptionPlan
.sendtable @target Send a formatted table to a phone number or group from manually entered data. Free
.sendgtable @target Fetch a public Google Sheet and send it as a formatted table. Premium
.stats Display bot statistics in a formatted table. Free

Carousel Messages

Send interactive WhatsApp carousel cards with images, titles, descriptions, and clickable action buttons — URL links, copy-to-clipboard codes, phone calls, or quick replies. Perfect for product listings, promotions, and rich content sharing.

Basic Carousel — .sendcarousel

whatsapp

|Hello Bro 👋|OURIN BOT|Quoted Text
|Netflix Premium
|Watch Unlimited Movies & Shows
|Footer Text|https://image-url.com/img.jpg
|url,🌐 Visit,https://netflix.com
|copy,📋 Copy Code,NET2026
|reply,🚀 Reply,Order Now

✅ Carousel sent!

Button Types

TypeSyntaxAction
url |url,Label,https://... Opens a URL in the browser.
copy |copy,Label,TEXT Copies TEXT to clipboard on tap.
call |call,Label,+91xxxxxxxxxx Initiates a phone call.
reply |reply,Label,Response text Sends a quick reply message.

Google Sheets Carousel — .sendgcarousel

Available in v2.5.0+. Generates a carousel automatically from a public Google Sheet — one card per row. Ideal for product catalogs and dynamic listings. (Premium feature)
